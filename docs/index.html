APAVS — Automated Pipeline and Visualisation System
HDip in Computer Science — Capstone Project
South East Technological University (SETU) · 2026

Abstract

APAVS automates the monitoring of load port qualification data across a fleet of semiconductor manufacturing tools. Currently, technicians manually export log files, run Excel macros, and visually inspect qualification plots for individual tools with no fleet-wide visibility. APAVS replaces this workflow with a PowerShell extraction pipeline that retrieves data from tool hard drives, parses raw sensor logs, calculates slot-level offsets, and feeds the results into a SQL Server database. A Power BI dashboard provides real-time pass/fail status, trend analysis, and fleet-wide visibility across the tool set, enabling engineers to identify drift before it impacts production.
